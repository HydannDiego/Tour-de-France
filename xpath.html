Les requêtes en xPath

Requête Xpath n°1

pour les documents XML respectant la DTD en 7 : fournir les contenus des éléments "documentation"

//documentation

Requête Xpath n°2

pour les documents XML respectant la DTD en 7 : fournir le premier élément "documentation"

//documentation[position() = 1]

Requête Xpath n°3

pour les documents XML respectant la DTD en 7 : fournir les éléments "documentation" n’ayant pas d’élément "lien"

//documentation[not(lien)]

Requête Xpath n°4

pour vos documents XHTML : fournir les valeurs des attributs href

//*/@href

Requête Xpath n°5

pour vos documents XHTML : compter le nombre d’images dans un document XHTML

//documentation
